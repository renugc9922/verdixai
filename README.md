# 🌱 VERDIXAI — AI Powered Crop Disease Detection

Smarter Crop Health with AI

VERDIXAI is an AI-powered crop disease detection platform designed to help farmers and agriculture users identify crop diseases from uploaded leaf images and receive treatment and prevention guidance.

---

## ✨ Features

- AI-powered crop disease detection
- Upload crop images
- Disease analysis and confidence score
- Treatment recommendations
- Prevention guidance
- Detection history
- AI Crop Assistant using Gemini API
- Premium dashboard UI
- Responsive modern interface

---

## 🚀 Installation

### Clone

```bash
git clone YOUR_REPOSITORY_URL
cd Verdix
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs at:

```plaintext
http://localhost:5173
```

---

### Backend

```bash
cd backend
npm install
npm run dev
```

Runs at:

```plaintext
http://localhost:5000
```

---

## 🔐 Environment Variables

Create:

```plaintext
backend/.env
```

Add:

```env
PORT=5000
GEMINI_API_KEY=your_api_key
```
